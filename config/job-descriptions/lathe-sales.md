# 职位描述 (Position Description)

1. 机电中专以上学历，对机电工程知识有了解
2. 二年以上车床销售或进口机销售经验
3. 有大客户开发及维护经验，对销售工作有热情
4. 负责STAR等车床设备的销售工作

# 必要条件 (Required Criteria)

- 机电中专及以上学历

- 2年以上车床销售或进口机销售经验
