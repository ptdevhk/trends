# 职位描述 (Position Description)

负责五轴加工中心的应用支持工作，包括编程、调试和客户培训。

# 必要条件 (Required Criteria)

- 大专及以上学历
- 机械/数控相关专业
- 3年以上五轴加工经验
- 熟练掌握五轴编程

# 优先条件 (Preferred Criteria)

- 熟悉多种五轴系统（FANUC/Siemens/Mazak）
- 有培训经验
- 有复杂零件加工经验

# 关键词 (Keywords)

五轴, 5轴, 加工中心, 编程, FANUC, Siemens, Mazak, 数控, CNC, 应用工程师
