# 职位描述 (Position Description)

1. 大专或以上学历，计算机、电子、自动化等相关专业
2. 1年以上软件开发经验，优质应届毕业生亦可考虑
3. C++语言基础扎实，熟悉MFC开发框架 或 C#语言基础扎实，熟悉WPF/WINFORM开发框架
4. 了解 RS232、TCP/IP、HTTP等通讯协议，有自动化设备相关软件开发经验者尤佳
5. 熟悉MySQL、SQLServer、Oracle等数据库中的一种，掌握SQL语言的基础知识

- 大专及以上学历
